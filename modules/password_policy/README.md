# Okta - R&T Terraform Package - Password Policies

This folder contains all the related files required to create password policies in Okta.
The module uses the following terraform resources:

1. [okta_policy_password](https://registry.terraform.io/providers/okta/okta/latest/docs/resources/policy_password)
2. [okta_policy_rule_password](https://registry.terraform.io/providers/okta/okta/latest/docs/resources/policy_rule_password)

### Description

This module is used to create all the password policies defined in the `password_policies.json` file.
